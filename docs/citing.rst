Citing and Acknowledgements
===========================

If cTreeBalls contributes to a publication, cite the software repository and
the associated scientific work identified by the project:

* Mario A. Rodriguez-Meza and contributors, `cTreeBalls
  <https://github.com/rodriguezmeza/cTreeBalls>`_.
* Abraham Arvizu et al., `arXiv:2408.16847
  <https://arxiv.org/abs/2408.16847>`_.

For reproducibility, record the cTreeBalls commit hash, compiler, active
Makefile settings, thread count, search method, input-catalog provenance, and
generated used-values parameter file.

The project acknowledges methods, software, and data infrastructure including
Zeno, Gadget-2, CUTE, Numerical Recipes, GNU Scientific Library, CLASS,
CFITSIO, and HEALPix.  Cite those projects when they are material to the
analysis.

The repository also acknowledges Abraham Arvizu, Alejandro Aviles, Juan Carlos
Hidalgo, Eladio Moreno, Gustavo Niz, Axel Romero Tisnado, Sofia Samario, and
other contributors and testers named in the project README.
