

To test the code:

$ time ../cballs options=post-processing script="python plot2pcf.py"

It will run the code, make a simple cube of random points, and as a post-process plot the 2pcf and save it as a pdf file.


time ../cballs parameters_test_T512_bin

Computation time is about a 3 minutes.

It uses a xyz catalog of points on the celestial sphere with kappa values. Total number of points is 3145728.
